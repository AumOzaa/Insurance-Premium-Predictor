# Insurance-Premium-Predictor

A machine learning project that predicts the insurance premium a person might be charged based on various attributes like age, BMI, smoking status, and more.

Features

Data preprocessing and cleaning

Exploratory Data Analysis (EDA)

Outlier Detection

Feature Scaling and Encoding

Model Training (Linear Regression)

Model Evaluation (RÂ² Score)

Deployment-ready prediction function



This project uses the [Medical Cost Personal Dataset](https://www.kaggle.com/datasets/mirichoi0218/insurance) which includes:

age: Age of primary beneficiary

sex: Gender of the individual

bmi: Body Mass Index

children: Number of children covered by health insurance

smoker: Smoking status

region: Residential region in the US

charges: Individual medical costs billed by health insurance
